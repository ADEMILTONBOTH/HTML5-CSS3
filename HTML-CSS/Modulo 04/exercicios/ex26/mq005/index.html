Testando Midia Quiries

Aqui estão os cinco recursos do freeCodeCamp desta semana que valem seu tempo:

1. Meu amigo Andrew Brown é um CTO que passou em praticamente todos os exames de certificação de nuvem existentes. Poucas semanas depois de o GitHub publicar seus novos certificados oficiais, Andrew já estudou, passou no exame e preparou este curso para ajudá-lo a fazer o mesmo. Se você está pensando em obter um certificado profissional para demonstrar sua proficiência em Git e GitHub, este curso é para você. (curso de 10 horas no YouTube): https://www.freecodecamp.org/news/pass-the-github-foundations-certification-course/

2. Entre muitos dos meus amigos desenvolvedores web, um novo conjunto de ferramentas está surgindo como padrão: Tailwind CSS, Next.js, React e TypeScript. E você pode aprender tudo isso codificando em casa com este curso para iniciantes. Você usará cada uma dessas ferramentas para criar seu próprio aplicativo de previsão do tempo. (curso de 2 horas no YouTube): https://www.freecodecamp.org/news/beginner-web-dev-tutorial-build-a-weather-app-with-next-js-typescript/

3. E se você quiser praticar ainda mais JavaScript, este tutorial é para você. Você codificará sua própria versão jogável no navegador do clássico jogo "Gorillas" de 1991, onde dois gorilas jogam bananas explosivas um no outro. Você usará JavaScript para a lógica do jogo e o loop principal do jogo. E você usará CSS e HTML Canvas para gráficos e animação. (leitura de 2 horas): https://www.freecodecamp.org/news/gorillas-game-in-javascript/

4. Deep Learning é uma abordagem profundamente útil para treinar IA. E se você quiser trabalhar na área de Machine Learning, este curso ensinará como responder 50 das perguntas mais comuns em entrevistas de emprego para desenvolvedores de Deep Learning. Você aprenderá sobre arquitetura de rede neural, funções de ativação, retropropogação, gradiente descendente e muito mais. (curso de 4 horas no YouTube): https://www.freecodecamp.org/news/ace-your-deep-learning-job-interview/

5. Meus amigos Jess e Ramón estão iniciando um novo grupo de seu bootcamp disponível gratuitamente na sexta-feira, 9 de fevereiro. Você pode se juntar a eles e trabalhar com a nova certificação de algoritmos JavaScript e estruturas de dados orientada a projetos do freeCodeCamp. Esta é uma ótima maneira de expandir suas habilidades ao lado de uma comunidade gentil e solidária. (leitura de 8 minutos): https://www.freecodecamp.org/news/free-webdev-and-js-bootcamps/

2024 está em pleno andamento. Não acredito que já é fevereiro. A comunidade freeCodeCamp começou a funcionar e estamos trabalhando duro em vários novos cursos de matemática, programação e ciência da computação para você. Nossa instituição de caridade é incrivelmente eficiente – entregando bilhões de minutos de aprendizado todos os anos com um orçamento irrisório. Mas poderíamos fazer ainda mais se tivéssemos mais recursos. Por favor, considere apoiar nossa missão: https://www.freecodecamp.org/donate

Citação da semana: “Git e eu temos um relacionamento sério, mas às vezes isso me incomoda.” - Cassidy Williams, engenheira de software

Até a próxima semana, boa codificação.
